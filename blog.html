Синтаксис CSS:

CSS стилі складаються із списку правил. Кожне правило має один або багато селекторів.

Синтаксис такий: спочатку пишеться селектор, потім в фігурних дужках вказуються всі його властивості і через двокрапку їх значення.

Як працює CSS?

14 Грудня 2016 Автор admin CSS> CSS

В цьому уроці ви створите свою першу таблицю стилів / style sheet. Ви дізнаєтеся про основи базової моделі CSS і про те, які коди необхідно використовувати для CSS в HTML-документі. Багато властивостей, використовувані в Cascading Style Sheets (CSS), аналогічні властивостям HTML. Таким чином, якщо ви використовуєте HTML для дизайну сторінок, ви, напевно впізнаєте багато кодів. Подивимося на конкретному прикладі.
